Scripted: Forms and Troubleshooting

DO NOW - Step 1

  1. Install the Postman Chrome Extension
  2. Launch Postman
    1. Shortcut: Open a new browser tab and type in chrome://apps then click the Postman icon
  3. Use Postman to send a GET request to the following URL: http://scripted-forms-mdepuy939679.codeanyapp.com/process-form.php
  4. Check out the response from that URL.
    1. Write down the status code you received for this request.

DO NOW - Step 3

  1. Use Postman to send a POST request to the following URL: http://scripted-forms-mdepuy939679.codeanyapp.com/process-form.php
    1. Inside the 'Body' tab in Postman, add a form parameter called 'foo' with a value of 'bar'
    2. Did you get a 200 status code? If not, you have a bug in your request!
